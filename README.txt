使用方式：雙擊「一鍵產生最新預測.bat」，會先取得官方最新開獎，再重算與同步本機手機版。

手機雲端獨立版：site 資料夾可直接發布至 GitHub Pages。
開獎後全自動更新：雙擊「安裝開獎後全自動更新.bat」。
雲端排程：台灣時間週一至週六 20:40～23:50，每10分鐘自動檢查、重算、驗收及發布。
獨立看門狗：21:00、22:00、23:30及隔日07:00比對官方與公開頁；落後會自動修復並再次驗證。

資料檔：data\539.csv
輸出檔：reports\最新539科學預測戰報.html、最新結果.json

特色：
1. 正式排名百分之百使用逐期擴展全歷史資料庫：頻率平衡、轉移反向校正、
   同型反向校正與遺漏反向校正；短期模組正式權重固定為零。
2. 每期真實搜尋全部286組錨定權重：最近360期三折校正，再以更早長歷史九段複驗；
   末段360期完全隔離，不得參與權重或滾動幅度選擇。
3. 滾動幅度由隔離期以前資料比較6種候選後鎖定；末段360期逐期先封存預測、
   開獎後才依錯誤模組更新下一期權重，最終權重用於下一期最強獨隻。
4. 組合限制奇偶、大小、區間、連號與重疊，提升多組覆蓋分散度。
5. 蒙地卡羅加權抽樣生成組合；固定資料會產生相同結果，方便驗算。
6. 移除排名完全重複的貝葉斯率及近期反向的共現關聯；同分號碼以當時期別公平破同分。
7. 每期固定產出並公開1中1主選；回測數據照實顯示，不得用門檻把號碼停發或清空。
8. 後段號碼只稱「排序後段」；高分未穩定勝過低分時，禁止稱為低機率或不中。
9. 每期開獎前封存完整39碼排序、逐號模組貢獻、全歷史資料庫指紋與封存雜湊；
   命中紀錄只採唯一正式封存，禁止測試版本或事後換號混入實戰歷史。
10. 開獎後逐項列出實際號碼原排名、主選與前5結果、錯誤模組鑑別差及權重調整；
    缺少檢討、滾動重算或證據雜湊時，整體更新必須失敗。
11. 發布前、發布後與獨立看門狗三層全面驗收；戰報可見文字只准中文。
