台灣 539 全新科學預測系統（2026-07-17）

使用方式：雙擊「一鍵產生最新預測.bat」。

手機雲端獨立版：site 資料夾可直接發布至 GitHub Pages。
開獎後全自動更新：雙擊「安裝開獎後全自動更新.bat」。
雲端排程：.github\workflows\auto-update.yml，台灣時間週一至週六 20:40、20:50、21:00 自動重試。

資料檔：data\539.csv
輸出檔：reports\最新539科學預測戰報.html、最新結果.json

特色：
1. 全球公開常見方法集成：多視窗頻率、Beta-Binomial 貝葉斯平滑、
   指數時間衰減、遺漏、動量、均值回歸、共現網路、一期轉移特徵。
2. 時間順序滾動回測，不使用未來資料。
3. 多模型競賽，自動比較候選權重並只採用回測較佳者；無效模組不硬加。
4. 組合限制奇偶、大小、區間、連號與重疊，提升多組覆蓋分散度。
5. 蒙地卡羅加權抽樣生成組合；固定資料會產生相同結果，方便驗算。
6. 13類特徵、7組候選集成、360期逐期樣本外盲測；只採用盲測最高模型。
7. 不允許偷看未來資料或用過度擬合宣稱零誤差；未達門檻一律標示研究觀察。

重要：今彩 539 是隨機遊戲，每一組合法號碼的理論機率相同。本工具只供統計研究與娛樂，不能保證中獎或獲利。請勿借貸投注，並設定固定娛樂預算。
